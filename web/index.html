About

MEMBRA Instant Proof Chain allows you to contribute to the network by simply staying on this page. Your browser will perform mining work, helping secure the network while you browse. This is a browser-based node that can run on any device including iPhones and Android phones.

Key Innovation: In this blockchain, sending a transaction is considered mining work, and the receiver optionally pays gas. By mining here, you're helping process transactions and secure the network.
